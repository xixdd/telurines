# Project Export in Huashijie Pro

Project export refers mainly to a file that contains all the layers and settings of a project.

Examples: Clipstudio (.clip); Photoshop (.psd) or (.ipv) from Ibis Paint

---
In this case, the file format is .hsj

It's worth clarifying that you can also export and import as .psd, but I'll focus on the Huashijie-specific case, so we'll use its native file format. This also means that this file type is not compatible with other programs, at least those outside of China.

___

As I've mentioned multiple times, most users won't have access to cloud storage, so this would be a good replacement.

___

___
___

### Step 1

First, go to the [export menu](readerhsp.html?file=saving.md) and choose the option that says <mark>HSJ草稿</mark>.
![1](images/imgexp/1.png)
___

### Step 2

Now select the folder icon.
![2](images/imgexp/2.png)

___
___

### Step 3

When this appears, click the text in the middle and give it whatever name you want. After that, confirm with the buttons you should already know.

![3](images/imgexp/3.png)
![4](images/imgexp/4.png)

---

### Step 4

This will be saved in your files. Look for the folder "<mark>HuashijiePro</mark>" and then "<mark>exportDrafts</mark>".

  HuashijiePro/exportDrafts/file.hsj

![5](images/imgexp/5.png)
___

There you'll see your file with the <mark>.hsj</mark> extension.
![](images/imgexp/6.png)
---

### Step 4.1

Select it, then tap "open with" and look for Huashijie — and you're done.

![](images/imgexp/7.png)

___
___

## Import Option #2

### Step 5

If you want an easier way, just open the app directly and in the top right corner you'll see <mark>3 dots</mark>. Tap them to open a menu, then <mark>choose option number 2</mark>.

![](images/imgexp/8.png)

___

### Step 6

This will open a media picker. First go to the second option at the top and you'll see all your files.
Then find and select the folder "<mark>HuashijiePro 画世界Pro</mark>".

![](images/imgexp/9.png)

___

### Step 7

Search again for "<mark>exportDrafts</mark>" and you'll find your file.

![](images/imgexp/10.png)
![](images/imgexp/11.png)

___

Once done, your project will be exactly as you left it, with all layers and settings intact.

![](images/imgexp/last.png)

___

Also, the next time you go through this process, when you open the media picker from <mark>Step 6</mark>, you'll see the folder you need directly.

![](images/imgexp/12.png)
