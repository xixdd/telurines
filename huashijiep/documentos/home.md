# Home Screen

Here I'll go over the things you can do on the home screen — this is one of the easier ones.

The topics we'll cover are:

- Delete, Rename, Copy, and Export
- Create folders and Sort
- Import projects
- Recently Deleted


___
___

First of all, I need to tell you that you cannot log into the app. When you open it, you'll see icons at the top for a magnifier, your profile, and a cloud. All of them require you to log in, so just forget about them.

## Delete, Rename, Copy, and Export

This is not complicated at all. Just hold down on any project you want and the icons will appear right there.

![](images/imgt1/menuhold.png)

Everything is pretty intuitive, but I'll name them again anyway.

From left to right:

 1. Delete
 2. Rename
 3. Copy
 4. Move (covered in the next section)
 5. Export

And that's it — works just like any other app you know.


___
___



## Create Folders and Sort

For this, go to the three dots in the top right corner.
There you'll see a short list of options, which from top to bottom are:

 1. Create folder
 2. Import
 3. Select
 4. Sort
 5. Recently deleted
 6. User guide (in Chinese)
 
![](images/imgt1/lista.png)

 
I'll explain everything, but folders first.

After selecting that option, give it a name and it's created.
Now, once inside your folder, you can create more folders. However, you cannot directly add projects from within the folder.

So you need to go back out and hold down on the project you want to save there. Then you'll use what was covered in the previous section: move.

![](images/imgt1/menuhold.png)

There you'll have the option to cancel (top right) or continue (bottom center).

___

### Sort

This may not matter to many people, especially if you don't have too many projects yet.
What it does is sort your projects either by date or name — meaning you'll see the most recent ones first, or in alphabetical order, depending on what you choose.

Just go to the 3 dots again and pick option number 4, then choose by date or by name, which are options 1 and 2 respectively.

![](images/imgt1/orden.png)

<mark>Note: Folders always appear at the top regardless of the sort order.</mark>

## Import Projects

This is the workaround for not being able to use cloud storage, though it's a bit more manual.

This is the second option in the dropdown list at the top right corner mentioned earlier, which I'll refer to as "list" or "options list."

It will open a file picker where you simply find your previously exported projects and import them.

![](images/imgt1/import.png)

___

Now, how do you export a project?

Here I'm not talking about exporting your finished drawing, but the editable file with all your layers.
For this you need to go to the [export menu](readerhsp.html?file=saving.md) and choose the one that says <mark>"HSJ草稿"</mark>.

![](images/imgt1/export.png)

Depending on how heavy your project is (more layers make it heavier), it may take more or less time to compress and export.

Of course, this is for when you switch devices — you save your HSJ file, share it to the new device, run the import process, and you're good to go.

Another thing worth mentioning is that you can also export as psd and tiff, which might mean you can also import files in those formats.

I've never tried it myself, but it's worth doing — you could even try with ipv files (ibispaint).

## Recently Deleted

This is one of the few apps that offers this, and it's especially useful here since you're more likely to accidentally delete a project, at least at first.

Go to option number 5 on the list and you'll see your deleted projects. Keep in mind that <mark>they stay here for 7 days before being permanently deleted.</mark>

There you'll have another dropdown list in the same position. The first option is to permanently delete everything, and the second is to restore. If you choose the latter, select the projects you want to recover and use the buttons <mark>you should have already learned</mark> to continue or cancel.

![](images/imgt1/er.png)

### Others

You may have noticed I didn't explain all the options in the list — that's because they're honestly a bit useless, but I'll briefly cover them here anyway.

___

#### Select
Option number 3 in the dropdown list. It's just another "shortcut" to select projects. I don't find it useful since you can already do it without opening the list, and it doesn't add anything new.
___
#### User Guide
This is the last option in the list — the name says it all.
The reason I don't recommend it is that it's not very descriptive, and it opens in an in-app browser, so to translate it you'd have to screenshot each part, which is inefficient.

##### New
This only appears after an update — it looks like a lightbulb icon and shows what was added in the update, but it's also in Chinese.

![](images/imgt1/final.png)
